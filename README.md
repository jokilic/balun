# Balun

A simple Flutter application for football scores.

## API Documentation
[Link] (https://www.api-football.com/documentation-v3)

# Screens

### `FixturesScreen`

* Show fixtures by date

### `PlayerScreen`

* Player from ID and season
* Injuries from player and season
* Transfers player
* Sidelined player
* Trophies player

### `CoachScreen`

* Coach from ID
* Sidelined coach
* Trophies coach
